# 望潮 Aurum｜顶层产品规划

更新时间：2026-07-15

## 一句话定位

望潮不是普通行情站，而是一个面向中文用户的**每日投资答案系统**：每天先看今日结论，再看自己的持仓，最后进入美股、港股打新、聪明人持仓和 A 股收息的完整研究链。

## 产品原则

1. **像抖音一样简单上手**：首页不要求用户先理解指标，先给结论和下一步。
2. **像 DeepSeek 一样简洁**：一屏一个问题，避免把研究过程全部堆给用户。
3. **像 Claude 一样美观实用**：克制、清晰、可阅读，所有视觉元素都服务于判断。
4. **数据可信优先于页面完整**：没有真实、可验证的数据时保持空白，不使用虚拟新股、静态价格或假精确结论补位。

## 核心用户

- 默认体验：投资新手也能直接理解。
- 核心留存与付费用户：已有港美股账户、已有持仓、有基础投资经验的人。

## 北极星指标

第一阶段的唯一第一目标是：**让用户每天愿意打开一次。**

建议用三个指标验证：

- 7 日内打开 ≥3 天的用户占比。
- 添加至少 1 只持仓的用户占比。
- 从首页进入具体标的或频道详情的点击率。

## 业务优先级

1. 美股投资。
2. 港股打新。
3. 聪明人持仓。
4. A 股收息。

四条业务线同时保留，但资源投入、首页排序和迭代速度按上述顺序执行。

## 核心使用闭环

1. 用户被微信群或站内提醒唤回。
2. 首页 30 秒看完“今日结论”。
3. 查看“我的持仓”是否触发观察、风险或止盈条件。
4. 进入频道信息流或具体标的页查看完整依据。
5. 形成自己的动作，第二天回来验证。

## 首页信息架构

1. 当前时段：早间、盘中、晚间或美股时段自动切换。
2. 今日结论：默认三条，优先美股、港股打新、聪明人披露。
3. 我的持仓：第一阶段手动添加代码、成本和数量，数据只保存在本地。
4. 四个投资入口：按业务优先级排列。
5. 免费与内测版边界。

频道内部继续使用连续信息流：一句结论 → 三个依据 → 一个风险 → 深入查看。

## 免费版与内测版

### 免费版

- 一句话结论。
- 三个关键依据。
- 一个主要风险。
- 数据日期与来源。

### 内测/会员版

- 买入参考区间。
- 止盈与止损参考区间。
- 触发升级、降级或继续等待的条件。
- 主标的与替代标的比较。
- 周度复盘与历史验证。

价格区间必须基于已验证的数据与规则，不承诺收益，不把未验证数字展示为确定答案。

## 数据生产链

第一阶段：自动数据 → 固定模型 → 人工复核 → 公开展示。

成熟阶段：自动数据 → 自动异常检测 → 模型生成初稿 → 高风险结论人工复核 → 自动发布。

所有公开数据必须具备：来源、时间、验证状态。港股新股不得使用虚拟样本；美股未验证价格区间必须隐藏；A 股分红计算必须使用真实每股分红数据。

## 版本规划

### P0｜已在本次首页重构落地

- 今日结论首页。
- 时段自动切换。
- 手动持仓与成本价。
- 持仓观察提示。
- 四入口按优先级重排。
- 数据失败时不输出静态替代答案。
- 原完整工具保留为 `legacy.html`。

### P1｜下一步

- 持仓与具体标的详情路由直接打通。
- 站内提醒记录与已读状态。
- 首页行为埋点：打开、添加持仓、详情点击、次日回访。
- 微信群每日卡片文案自动生成。
- 免费版与内测版权限状态正式接入。

### P2｜验证留存后再做

- 持仓截图识别。
- 登录与跨设备同步。
- 邮件、浏览器或微信服务通知。
- 个性化日报和周报。
- 更自动化的人工复核工作台。

## 明确不做

- 不把首页做成行情大屏。
- 不同时展示大量指标。
- 不新增未经验证的股票、投资人或榜单。
- 不为了页面“看起来完整”而加入虚拟数据。
- 不在没有留存验证前开发复杂通知、社交或交易功能。
