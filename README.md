## 产品定位

望潮不是普通行情站，而是面向中文用户的「港美股投资答案工具」。

核心原则：

1. 先给结论：用户先知道值不值得看。
2. 再给风险：不把不确定性包装成确定性。
3. 最后给入口：港股打新、美股机会、A股收息、聪明人持仓各自分流。

## 增长转化设计

`growth.html` 用于对外招募内测用户和解释会员价值：

- 明确适合用户：已有港美股账户、有基础投资经验、关心AI主线/高股息/打新。
- 明确免费/付费边界：免费负责建立信任，付费负责完整判断链、买入区间、替代标的和周报。
- 提供可复制文案：包含社群版和朋友圈版，方便微信群、朋友圈、私信分发。
- 保留合规边界：工具只做信息整理、数据分析和研究辅助，不构成投资建议。

## 页面层级

- 一级：首页只保留港股打新、美股投资、A股收息三个入口。
- 驾驶舱页：解释产品结构、免费/付费边界和后续升级方向。
- 增长页：解释谁适合用、怎么内测、怎么传播、为什么值得持续使用。
- 二级：港股按值得打、谨慎打、不建议、已结束分类；美股展示七姐妹、热度前三和聪明人持仓；A股按买入、等待、回避分类。
- 三级：只展示具体标的和一句话结论，点击后再进入完整答案。
- 四级：港股展示申购结论、暗盘、首日、五日卖出答案和站内资料；美股展示买入、止盈、止损、基本面和聪明人持仓；A股展示收息评级、价格和分红测算。
- 五级：展示评分拆解、策略边界、历史回溯、性价比指数和持仓事实与解读。

深层研究入口：

- 港股完整数据回溯：`#/hk/backtest`
- 美股热度前十：`#/us/hot-10`
- 美股性价比指数：`#/us/value`
- 单一标的深度分析：在港股、美股、A股及聪明人详情页进入。
