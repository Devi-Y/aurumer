## 产品定位

望潮不是普通行情站，而是面向中文用户的「港美股与黄金投资答案工具」。

核心原则：

1. 先给结论：用户先知道值不值得看。
2. 再给风险：不把不确定性包装成确定性。
3. 最后给入口：港股打新、美股机会、A股收息、黄金投资、聪明人持仓各自分流。

## 增长转化设计

`growth.html` 用于对外招募内测用户和解释会员价值：

- 明确适合用户：已有港美股账户、有基础投资经验、关心AI主线/高股息/打新。
- 明确免费/付费边界：免费负责建立信任，付费负责完整判断链、买入区间、替代标的和周报。
- 提供可复制文案：包含社群版和朋友圈版，方便微信群、朋友圈、私信分发。
- 保留合规边界：工具只做信息整理、数据分析和研究辅助，不构成投资建议。

## 页面层级

- 一级：首页以 Bento 布局依次展示港股打新、美股投资、A股收息、黄金投资和聪明人持仓五个入口；黄金固定第 4，聪明人持仓固定最后。
- 每日驾驶舱：独立展示今日结论和仅保存在当前浏览器的个人持仓，不改变主站三级入口结构。
- 驾驶舱页：解释产品结构、免费/付费边界和后续升级方向。
- 增长页：解释谁适合用、怎么内测、怎么传播、为什么值得持续使用。
- 二级：港股按值得打、谨慎打、不建议、已结束分类；美股展示七姐妹和热度前三；A股按买入、等待、回避分类；聪明人持仓独立分为港股 3、美股 5、A股 3。
- 三级：只展示具体标的和一句话结论，点击后再进入完整答案。
- 四级：港股展示申购结论、暗盘、首日、五日卖出答案和站内资料；美股展示买入、止盈、止损和基本面；A股展示收息评级、价格和分红测算；聪明人详情展示业绩口径、WHY、HOW 与具体持仓。
- 黄金：二级先选当前答案、价格位置、驱动与风险；深层页再展示国际金、上海金、实际利率、美元、金融条件与 CFTC 持仓。
- 五级：展示最终分数与排名、历史表现、性价比指数、情景压力测试和持仓事实与解读。

深层研究入口：

- 港股完整数据回溯：`#/hk/backtest`
- 美股热度前十：`#/us/hot-10`
- 美股性价比指数：`#/us/value`
- 单一标的深度分析：在港股、美股、A股及聪明人详情页进入。

## 多端使用

- 网页：固定 GitHub Pages 地址，电脑和手机浏览器自动响应式适配。
- 手机桌面：站点提供 `manifest.webmanifest` 和 Service Worker，可作为 PWA 添加到主屏幕。
- 微信小程序：`miniprogram/` 提供从入口、分类、列表到完整详情的纯原生链路，不打开外部网页。
- 微信发布前唯一必须人工完成的步骤：填写独立的望潮小程序 AppID，上传并提交审核；无需购买服务器或域名。
- 分享：港股、美股、A股和聪明人详情页支持系统分享；不支持系统分享时自动复制短链接。
- 短链接：每次数据刷新会自动生成首页、分类页和全部标的的静态入口，未执行 JavaScript 的访问端也能先收到有效页面。

## 公开展示边界

- 港股详情区分申购结论、暗盘、首日与五日结果，并展示同组分数和排名。
- 美股提供七姐妹、热度前三、热度前十和性价比指数；有数据时展示买入、止盈、止损研究参考价，并明确验证状态。
- A股已提供分红质量、价格层级、经营现金流、自由现金流、分红计算器、年度压力测试与五年累计现金流情景。
- 聪明人持仓按港股 3、美股 5、A股 3 展示候选池内收益排序、业绩口径、公开持仓、WHY、HOW 和披露边界；不同市场、币种、区间和份额不做全球绝对横比。
- 黄金同时展示国际金与上海金的结论和研究区间，并将实际利率、美元、流动性、持仓拥挤与本地溢价放进同一判断页。
- 公开页面不展示内部判断规则、权重、阈值或评分拆解，只展示最终分数、排名、结论、事实与风险。
- 公开单页只展示整理后的最终答案，浏览器不执行分析计算。
- 策略引擎、历史信号和审计记录保存在私有环境；自动任务只向本仓库发布脱敏后的最终快照与短链接。
- 所有结论都展示升级、降级或继续等待的条件，不承诺收益，也不使用占位价格。

## 自动验收

运行 `npm run audit` 会同时检查：

- 公开快照不含策略、模型、权重和内部验证字段。
- 港股、美股、A股、黄金和聪明人数据完整，价格答案与现金流字段可用。
- 92 个公开直达页、canonical、分享说明、分享图和 sitemap 完整。
- 一级到五级页面结构、移动端导航、加载状态、排名与深度内容没有被后续更新破坏。

页面数据仅用于信息整理、数据分析和投资研究辅助，不构成任何投资建议、收益承诺或买卖指令。
