# 望潮微信小程序

望潮小程序采用全原生答案链路，不打开外部网页：

1. 首页只显示港股打新、美股投资、A股收息三个入口。
2. 二级只显示港股四类、美股三类、A股三类结论入口。
3. 三级显示具体标的一句话结论、分数、排名和关键价格。
4. 四级完整分析直接在原生详情页展示资料、价格、持仓、风险和下一步动作。

小程序读取随版本打包的公开数据快照，不使用 `web-view`，也不在运行时请求外部网站。因此不需要购买域名、配置业务域名或配置 request 合法域名，用户不会在望潮小程序内部遇到“非微信官方网页”的继续访问确认页。

## 免费更新方式

网页公开数据更新后，在仓库根目录执行：

```bash
npm run sync:mini
npm run check:mini
```

`sync:mini` 会把 `data/live-snapshot.json` 中允许公开的字段写入 `miniprogram/data/live-snapshot.js`。内部策略、权重、阈值和回测过程不会进入小程序。

## 正式发布前的唯一人工配置

将 `project.config.json` 中的 `touristappid` 换成独立的望潮小程序 AppID，然后用微信开发者工具上传并提交审核。不要复用问岳的 AppID。

正式发布过程本身不需要购买服务器或域名；数据更新随小程序版本发布。
